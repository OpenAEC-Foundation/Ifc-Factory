IfcX v2 Schema - Architecture Overview

IFC5-compatible node-based JSON format with 2D CAD extensions

1. Document Structure

header

File metadata and project info
{ "ifcxVersion": "2.0", "id": "uuid", "author": "...", "timestamp": "ISO8601", "units": { "length": "mm", "angle": "rad" }, "coordinateSystem": { "epsg": 28992 } }

imports

External schema references (IFC5-style)
[ {"uri": "ifc@v5a.ifcx"}, {"uri": "usd@v1.ifcx"}, {"uri": "nen47@v1.ifcx"} ]

data

Flat array of IfcxNode objects
[ {"path": "guid-1", "children": {"Wall": "guid-2"}, "attributes": { "bsi::ifc::class": {"code": "IfcProject"} }}, {"path": "guid-2", "attributes": { "ifcx::geom::line": {"points": [[0,0],[100,0]]} }} ]

media

Embedded raster images / binary data
{ "img-001": { "mimeType": "image/png", "width": 1920, "height": 1080, "data": "base64..." }, "img-002": { "uri": "photo.jpg" } }

Concept
v1 (DXF-in-JSON)
v2 (IFC5-native)

header, tables, blocks, entities, objects
header, imports, schemas, data[], media

{"type": "LINE", "handle": "A1"}
{"path": "uuid", "attributes": {"ifcx::geom::line": ...}}

Paper space
LAYOUT + VIEWPORT objects

scaleX, scaleY, rotation (degrees)

References

UUID paths + {"ref": "uuid"}
IFC integration
None (separate format)
bsi::ifc::class, bsi::ifc::prop::*
SVG/CSS styling
None

4. Attribute Namespace Map

ifcx::geom::* (Geometry)

  • line2 points
  • polylineN points + closed
  • compositeCurveline/arc segments
  • circlecenter + radius
  • trimmedCurvearc (center, angles)
  • ellipseaxes + rotation
  • bsplineNURBS curve
  • meshvertices + faces
  • solidBRep/ACIS data
  • pointsingle point

ifcx::annotation::* (Annotation)

  • textplain text
  • richTextformatted spans
  • dimensionlin/ang/rad/dia/ord
  • leaderpath + arrowhead
  • toleranceGD&T frames
  • tablerows/cols/cells
  • tagelement tag (INB)
  • symbolplaced symbol

ifcx::style::* (Presentation)

  • curveStylecolour + width + dash
  • fillStylefill colour + pattern
  • textStylefont + size + weight
  • colourRGBA (0-1)
  • dashPattern[vis, gap, ...]
  • lineWeightmm
  • opacity0-1

ifcx::hatch::* (Hatching)

  • patterncustom line pattern
  • solidsolid colour fill
  • gradientgradient fill
  • svgSVG pattern ref
  • nen47NEN 47 standard
  • boundaryouter + inner refs

ifcx::sheet::* (Layouts)

  • papersize + margins
  • titleBlockblock ref
  • viewportview into model
  • viewTitlename + scale
  • plotSettingsprint config

ifcx::svg::* (SVG/CSS)

  • classCSS class names
  • strokeline colour
  • strokeWidthline width
  • fillfill colour/pattern
  • fontFamilytext font
  • markerEndarrow marker

bsi::ifc::* (IFC Integration)

  • classIfcWall, IfcSlab, ...
  • materialmaterial ref
  • guidIFC GlobalId
  • prop::Heightproperty value
  • prop::Nameelement name

ifcx::geo::* (GIS)

  • crsEPSG + WKT
  • positionlat/lon/alt
  • featureGeoJSON geometry

ifcx::image::* (Raster)

  • rastermediaId + placement
  • wipeoutmasking boundary
